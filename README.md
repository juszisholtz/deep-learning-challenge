# deep-learning-challenge

This repository contains the code, analysis, and model for the Alphabet Soup Charity project, where we analyze and predict charity data.

Files

    Starter_Code/AlphabetSoupCharity.keras: This file contains the trained model from the project, saved in Keras format.
    
    Starter_Code/My_Report.md: A markdown file detailing the analysis and findings of the project.
    
    Starter_Code/Starter_Code.ipynb: The Jupyter notebook containing the code used for data processing, model building, and evaluation.
